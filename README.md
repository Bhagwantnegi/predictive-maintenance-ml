# 🚗 Predictive Maintenance for Fleet Vehicles

## 📌 Overview
This project develops a machine learning model to predict potential vehicle failures using sensor and operational data. The goal is to enable proactive maintenance and reduce downtime in fleet operations.

---

## 🎯 Problem Statement
Fleet vehicles often experience unexpected failures, leading to:
- Increased operational costs  
- Unplanned downtime  
- Safety risks  

This project aims to **predict failures before they occur** using data-driven techniques.

---

## 🧪 Approach

### 1. Data Preparation
- Cleaned and structured sensor data  
- Handled missing values  
- Selected relevant features  

### 2. Exploratory Data Analysis (EDA)
- Identified trends and correlations  
- Analyzed sensor behavior before failures  

### 3. Model Development
- Built classification models using:
  - Random Forest  
  - Logistic Regression  
- Evaluated performance using:
  - Accuracy  
  - Precision  
  - Recall  
  - Confusion Matrix  

---

## ⚙️ Tech Stack
- Python  
- Pandas, NumPy  
- scikit-learn  
- Jupyter Notebook  

---

## 📊 Key Results
- Successfully predicted high-risk vehicles  
- Demonstrated ability to identify failure patterns  
- Enabled proactive maintenance recommendations  

---

## 💡 Business Impact
- Reduced risk of unexpected breakdowns  
- Improved operational efficiency  
- Enabled data-driven maintenance decisions
